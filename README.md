# 💰 Automação de Controle Financeiro com Python

## 📌 Sobre o Projeto

Este projeto é uma pequena automação desenvolvida em **Python** que gera uma planilha de **controle financeiro** automaticamente.

O sistema permite adicionar receitas e despesas via terminal e, ao final, calcula automaticamente:

- Total de receitas  
- Total de despesas  
- Saldo final  

A planilha é criada no formato `.xlsx` utilizando a biblioteca **openpyxl**.

---

## 🚀 Tecnologias Utilizadas

- Python 3  
- OpenPyXL  

---

## 📂 Estrutura do Projeto

```

 ├── main.py
 └── controle_financeiro.xlsx (gerado automaticamente)
```
